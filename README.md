# IotDataTensor
A final class inheritance project which marks the beginning of my work with data structures (in this case tensors), and data processing using CSV shards.

This project was my submission for a bonus question submitted by my professor Dr. Yan Liu, currently teaching at Concordia University. It is based off of the classes BaseTensor.h, RankOneTensorType.h, and IotDataTensor.h which she authored and provided for every student in her class.
The entire assignment (including questions which are not included in this repository) covered topics of polymorphism, function/class templates, and an introduction to data structures (in this case tensors).

Please note:
- BaseTensor class was not authored by me but by Dr. Yan Liu.
- RankOneTensorType class was only **partially** authored by myself. The barebones were written by my professor Dr. Yan Liu. I was tasked with completing (delibaretely) missing components of the class, including constructors, proper destructors, loadData() overriden functions for both expected data type tensors (double & std::string).
- The same applies for the IotDataTensor class, it is based off of given code by my professor.
- Included is a .txt file showing exactly what was given as material, and the header file shows the completed class which does include my own work.
- The specific instructions pdf for this assingnment is not included in this repository as it is not my in intellectual property to distribute.
